# Envíos DosRuedas | EDR CORE - Sistema Canvas v4.0

## Propuesta Operativa Estratégica 2026

Este proyecto es una plataforma de presentación interactiva y de alta gama diseñada para **Envíos DosRuedas (EDR)**. El sistema, denominado **Canvas v4.0**, permite la visualización y exportación técnica de la propuesta operativa para el periodo 2026, enfocada en la logística de última milla en Mar del Plata.

### 🚀 Características Principales

- **Diseño Industrial Premium**: Estética "Dark Gold" con una paleta de colores corporativa (Azul Profundo y Oro Industrial) optimizada para una percepción de marca sólida y profesional.
- **Formato A4 Nativo**: Implementación de un sistema de slides que respeta estrictamente la proporción ISO 210x297mm (1:1.414), garantizando una exportación perfecta a PDF.
- **Arquitectura de 11 Páginas**:
  - **Identidad**: Portada estratégica y visión de negocio.
  - **Modelos Operativos**: Detalle de servicios Express (por demanda), Lowcost (ruteo diario) y Flex (MercadoLibre).
  - **Soluciones E-Commerce**: Planes de Fulfillment 3PL y Distribución 24hs.
  - **Gestión de Cuentas**: Sistema de Cuenta Corriente Flexible para Pymes.
  - **Análisis Comparativo**: Resumen ejecutivo y tabla comparativa de modelos.
  - **Soporte**: Resolución de excepciones y FAQs operativas detalladas.
  - **Cierre Estratégico**: Pasos de implementación y contacto directo.
- **Optimización Multi-dispositivo**: Responsive Design que adapta la experiencia de lectura de A4 en computadoras a una vista fluida en dispositivos móviles.
- **Print-Ready**: CSS configurado específicamente para la impresión limpia del documento, eliminando elementos de navegación innecesarios.

### 🛠️ Stack Tecnológico

- **Framework**: [Next.js 15+](https://nextjs.org/) (App Router)
- **Lenguaje**: [TypeScript](https://www.typescriptlang.org/)
- **Estilos**: [Tailwind CSS](https://tailwindcss.com/)
- **Componentes UI**: [Shadcn/UI](https://ui.shadcn.com/)
- **Iconografía**: [Lucide React](https://lucide.dev/)
- **Fuentes**: Montserrat (Headline), Manrope (Body), JetBrains Mono (Data Points)

### 📋 Estructura del Proyecto

```text
src/
├── app/              # Rutas y Layout principal
├── components/       # Componentes de UI y Core
│   ├── pages/        # Implementación individual de cada slide A4
│   └── ui/           # Librería de componentes Shadcn
└── lib/              # Utilidades y configuración de assets
```
